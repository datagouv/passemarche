# 2. APIs intégrées

Le tableau ci-après présente les **principaux services de données (API) intégrés à ce jour par Passe Marché** pour la récupération automatique des informations et justificatifs nécessaires à la constitution des candidatures.

Ce tableau a pour objectif de **donner de la visibilité** sur les sources mobilisées par Passe Marché, de **faciliter la compréhension du fonctionnement des automatisations**, et de permettre aux opérateurs économiques d’identifier, en toute transparence, les données susceptibles d’être récupérées lors d’une candidature.

| API                                            | Ce qui est récupéré / objectif                                                                                                                                                                                                                   | Lien                                                                                                          |
| ---------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------- |
| **INPI – Extrait RNE**                         | Raison sociale, activité principale, dirigeant, adresses (données officielles)                                                                                                                                                                   | https://entreprise.api.gouv.fr/catalogue/inpi/rne                                                             |
| **INSEE – Établissements (SIRET / catégorie)** | Données SIRET + catégorie d’entreprise                                                                                                                                                                                                           | https://entreprise.api.gouv.fr/catalogue/insee/etablissements                                                 |
| **URSSAF – Attestation de vigilance**          | Attestation URSSAF (PDF) + éléments utilisés pour certaines déclarations (ex. obligations sociales, OETH via mention)                                                                                                                            | https://entreprise.api.gouv.fr/catalogue/urssaf/attestation\_vigilance                                        |
| **DGFIP – Attestations fiscales**              | Attestation fiscale (PDF) prouvant la régularité fiscale                                                                                                                                                                                         | https://entreprise.api.gouv.fr/catalogue/dgfip/attestations\_fiscales                                         |
| **DGFIP – Chiffres d’affaires**                | CA global annuel sur plusieurs exercices                                                                                                                                                                                                         | https://entreprise.api.gouv.fr/catalogue/dgfip/chiffres\_affaires                                             |
| **INPI – Actes & Bilans**                      | Bilans disponibles (PDF) pour les exercices attendus                                                                                                                                                                                             | https://entreprise.api.gouv.fr/catalogue/inpi/rne/actes\_bilans                                               |
| **BODACC (source externe)**                    | Détection de situations pouvant constituer des **motifs d’exclusion** (ex. liquidation, interdiction de gérer) + justificatifs                                                                                                                   | https://www.bodacc.fr/pages/api-bodacc/ _(source externe)_                                                    |
| **INSEE – Statut ESS**                         | Indication ESS                                                                                                                                                                                                                                   | https://entreprise.api.gouv.fr/catalogue/insee/etablissements                                                 |
| **PRO BTP – Conformité cotisations retraite**  | Attestation PRO BTP (PDF) pour certains marchés                                                                                                                                                                                                  | https://entreprise.api.gouv.fr/catalogue/probtp/conformites\_cotisations\_retraite                            |
| **CIBTP – Attestations cotisations**           | Attestation CIBTP (PDF)                                                                                                                                                                                                                          | https://entreprise.api.gouv.fr/catalogue/cibtp/attestations\_cotisations\_conges\_payes\_chomage\_intemperies |
| **CNETP – Attestations cotisations**           | Attestation CNETP (PDF) (marchés travaux)                                                                                                                                                                                                        | https://entreprise.api.gouv.fr/catalogue/cnetp/attestations\_cotisations\_conges\_payes\_chomage\_intemperies |
| **FNTP – Carte professionnelle TP**            | Carte professionnelle Travaux Publics (PDF)                                                                                                                                                                                                      | https://entreprise.api.gouv.fr/catalogue/fntp/carte\_professionnelle\_travaux\_public                         |
| **Qualibat – Certifications bâtiment**         | Certification Qualibat (PDF)                                                                                                                                                                                                                     | https://entreprise.api.gouv.fr/catalogue/qualibat/certifications\_batiment                                    |
| **Qualifelec – Certificats**                   | Certification Qualifelec (PDF)                                                                                                                                                                                                                   | https://entreprise.api.gouv.fr/catalogue/qualifelec/certificats                                               |
| **ADEME – Certifications RGE**                 | Certifications RGE (PDF)                                                                                                                                                                                                                         | https://entreprise.api.gouv.fr/catalogue/ademe/certifications\_rge                                            |
| **OPQIBI – Qualifications ingénierie**         | Données de certification (dates, durée…)                                                                                                                                                                                                         | https://entreprise.api.gouv.fr/catalogue/opqibi/qualifications\_ingenierie                                    |
| **France Compétences / CARIF-OREF**            | Statut, dates, habilitations (former / organiser), etc.                                                                                                                                                                                          | https://entreprise.api.gouv.fr/catalogue/carif\_oref/certifications\_qualiopi\_france\_competences            |
| **Qualiopi**                                   | Types de certification Qualiopi                                                                                                                                                                                                                  | https://entreprise.api.gouv.fr/catalogue/carif\_oref/certifications\_qualiopi\_france\_competences            |
| **Actes et bilans**                            | Liste des actes et bilans d'une unité légale inscrite au répertoire national des entreprises (RNE). Seuls les actes et bilans publiques sont distribués. L'INPI peut potentiellement posséder des actes et bilans "partiellement confidentiels". | https://entreprise.api.gouv.fr/catalogue/inpi/rne/actes\_bilans                                               |

Note : les URL sont conservées telles qu’elles apparaissent dans la source.
