# 6. Renseigner les capacités économiques et financières

Les informations et champs qui vous sont présentés dans cette section dépendent directement du **paramétrage effectué par l’acheteur** pour le marché concerné.

Lorsque l’acheteur a choisi de demander des éléments relatifs aux **capacités économiques et financières**, Passe Marché affiche dans le formulaire les champs correspondants afin que vous puissiez :

* renseigner les informations demandées,
* et, le cas échéant, joindre les documents requis.

Par ailleurs, certaines informations nécessaires à l’analyse de la candidature peuvent être **récupérées automatiquement** par Passe Marché auprès d’administrations ou de sources de référence. Ces données sont alors **transmises à l’acheteur**, sans action requise de votre part.

À l’inverse, si l’acheteur n’a pas sollicité ce type d’informations pour le marché, cette section peut être absente ou limitée.

Passe Marché permet ainsi d’adapter le contenu du formulaire aux besoins spécifiques de chaque marché, en évitant de vous demander des informations inutiles.
