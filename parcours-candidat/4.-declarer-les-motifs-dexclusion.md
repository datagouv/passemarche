# 4. Déclarer les motifs d’exclusion

Dans la section consacrée aux motifs d’exclusion, une case unique vous permet d’attester sur l’honneur que votre entreprise n’est concernée par aucun motif d’exclusion.

Une information réglementaire est affichée sous cette case, à titre explicatif.

![](<../.gitbook/assets/image (10)>)

{% hint style="info" %}
Lorsque des informations sont détectées automatiquement

Dans certains cas, **des informations concernant votre entreprise peuvent être transmises automatiquement par une administration** dans le cadre du contrôle des motifs d’exclusion prévus par la réglementation.

Lorsque cela se produit, Passe Marché vous en informe de manière explicite au sein d’un encart dédié, afin de garantir une **transparence totale** sur les données prises en compte.
{% endhint %}

Il est possible **d’apporter des précisions ou de contester une information**

Les informations transmises automatiquement peuvent, dans certaines situations, être :

* **obsolètes**,
* **incomplètes**,
* ou **ne plus refléter votre situation actuelle**.

C’est pourquoi Passe Marché vous offre la possibilité, pour chaque information détectée, de :

* fournir une **explication écrite** (facultative),
* et, si nécessaire, de joindre un **document justificatif**.

Cette démarche vous permet d’apporter des éléments de contexte utiles à l’acheteur et de **prévenir toute interprétation erronée** de votre situation.

{% hint style="info" %}
Une démarche facultative, sans blocage du parcours

Il est important de noter que :

* la fourniture d’informations complémentaires est **entièrement facultative**,
* elle **n’est pas obligatoire pour poursuivre votre candidature**,
* dès lors que vous avez coché la case d’attestation sur l’honneur, vous pouvez continuer le formulaire même si vous ne souhaitez pas compléter ces informations.

Passe Marché a été conçu pour garantir que **votre candidature ne soit pas écartée automatiquement** sur la seule base d’une information détectée, sans vous avoir permis de vous exprimer.
{% endhint %}
