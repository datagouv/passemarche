# 7. Renseigner les capacités techniques et professionnelles

Les informations et champs qui vous sont présentés dans cette section dépendent directement du **paramétrage effectué par l’acheteur** pour le marché concerné.

Lorsque l’acheteur a choisi de demander des éléments relatifs aux **capacités techniques et professionnelles**, Passe Marché affiche dans le formulaire les champs correspondants afin que vous puissiez :

* renseigner les informations demandées,
* joindre les documents requis.

{% hint style="info" %}
Par ailleurs, certaines informations nécessaires à l’analyse de la candidature peuvent être **récupérées automatiquement** par Passe Marché auprès d’administrations ou de sources de référence.\
Ces données sont alors **transmises à l’acheteur**, sans action requise de votre part.
{% endhint %}
