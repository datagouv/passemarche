# 6. Consulter une candidature déposée

Une fois les candidatures déposées, **Passe Marché transmet l’ensemble des éléments de candidature vers votre plateforme éditeur**.

Vous pouvez alors **consulter directement sur votre plateforme éditeur** :

* les informations administratives et déclaratives transmises par les candidats,
* les données récupérées automatiquement lorsque cela est possible,
* les documents joints dans le cadre de la candidature.

Passe Marché garantit que les éléments transmis correspondent exactement aux exigences que vous avez définies lors de la configuration du marché, afin de vous permettre d’analyser les dossiers dans votre environnement habituel, sans rupture de parcours.
